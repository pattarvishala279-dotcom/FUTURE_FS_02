# Mini CRM Backend

## Description
This project is a Mini CRM backend application built using Node.js, Express.js, and MongoDB.
It helps manage client leads through REST APIs.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## Features
- Create and fetch leads
- MongoDB database connection
- REST API implementation

## How to Run
1. Open terminal inside backend folder
2. Run: npm install
3. Run: npx nodemon server.js
4. Open browser: http://localhost:5000/api/leads
